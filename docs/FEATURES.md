# 🌟 Características de AURA

## 🤖 Conversaciones con IA

### Chat Inteligente
- **Conversaciones empáticas** diseñadas específicamente para adultos mayores
- **Respuestas contextuales** que mantienen el hilo de la conversación
- **Interfaz simple y clara** con texto grande y botones accesibles
- **Eliminación de conversaciones** con confirmación de seguridad

### Tipos de Conversación
- **General**: Conversaciones casuales y de compañía
- **Salud**: Consejos y seguimiento de bienestar
- **Memoria**: Ejercicios para mantener la mente activa
- **Entretenimiento**: Historias, chistes y contenido divertido

## 📖 Diario Personal

### Funcionalidades del Diario
- **Entradas de texto** para registrar pensamientos y experiencias
- **Grabación de voz** con transcripción automática a texto
- **Estados de ánimo** para hacer seguimiento emocional
- **Historial completo** de todas las entradas guardadas
- **Búsqueda y filtrado** por fecha y contenido

### Transcripción de Voz
- **Tecnología OpenAI Whisper** para alta precisión
- **Optimizado para español** y acentos latinos
- **Procesamiento local seguro** de archivos de audio
- **Integración fluida** con el editor de texto

## 🔐 Seguridad y Privacidad

### Autenticación
- **Login seguro** con tokens de autenticación
- **Sesiones protegidas** con Laravel Sanctum
- **Validación de usuarios** en cada operación

### Protección de Datos
- **Base de datos local** SQLite para máxima privacidad
- **Encriptación de contraseñas** con bcrypt
- **Eliminación segura** de archivos temporales
- **No almacenamiento** de datos sensibles en la nube

## 💡 Filosofía del Proyecto

AURA está diseñado con el principio de **"Tecnología que acompaña, no complica"**. Cada característica está pensada para:

- **Reducir la soledad** en adultos mayores
- **Mantener la independencia** digital
- **Proporcionar compañía** cuando la necesiten
- **Respetar la privacidad** y dignidad
- **Ser fácil de usar** sin sacrificar funcionalidad
- **Adaptarse a diferentes niveles** de habilidad tecnológica
